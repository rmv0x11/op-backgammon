# op-backgammon

# Build:
`go build cmd/backgammon.go`  
`./backgammon`

# Methods

`POST /create_table` creates players table.  
`POST /add_player` adds a new player.  
`GET /get_players` get all players info.  
`GET /new_match` get stats about new match
